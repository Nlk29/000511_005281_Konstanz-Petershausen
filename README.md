# Konstanz-Petershausen

Das Modul Konstanz-Petershausen für Zusi 3 zeigt die Bahnstrecke Konstanz-Radolfzell von der Konstanzer Rheinbrücke bis zum mittlerweile ehem. Bf Reichenau(Baden).
